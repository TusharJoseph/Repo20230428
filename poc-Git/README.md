**Steps to setup Power BI development environment with Git Version Control.**
Create GitHub repository with default main branch
Power BI Workspace to link Git repository to this main branch
Create Feature1 branch and clone repository to local device using GitHub Desktop
Commit PBIP files to Feature1 branch
Create Feature2 branch and clone repository on another local device
Commit PBIP files to Feature2 branch
Create Pull request to merge changes and resolve conflicts from Feature1 branch to Feature2 branch
Create Pull request to merge changes from Feature2 branch to main branch
Power BI Workspace should be updated with latest PBIP changes.
